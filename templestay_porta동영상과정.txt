------------------------------------------
안녕하세요 템플스테이 포털 프로젝트 입니다.

작업환경  - java 1.*  - Spring 4.x  - Tomcat 8.0
             - MySQL 5.x  - MyBatis   - Gradle 3.x
적용언어   - java  - HTML5  - CSS
              - JavaScrpit    - jQuery

-----------------------------------
주된 기능으로 예약기능을 구현했습니다.

사찰이름으로 검색할 경우 해당 사찰의 목록이 출력됩니다.

지역 목록에는 해당 지역의 프로그램 수를
DB와 연동해 출력하게끔 구현했습니다.

지역, 테마, 날짜를 선택 후 찾는 경우에도
해당 사찰의 목록이 출력됩니다.

목록 페이지에서
사찰 이름과 지역, 테마, 날짜를 선택한 후 검색할 경우
해당 사찰의 목록이 출력됩니다.

지역만 선택후 검색할 경우 해당 지역에 소속된 
모든 사찰의 프로그램이 출력됩니다.

테마만 선택후 검색할 경우 해당 테마의 
모든 프로그램이 출력됩니다.

----------------------------

아무것도 선택하지 않을 경우 모든 프로그램이 출력됩니다.

날짜를 선택할 경우
해당 날짜와 DB의 예약가능일을 비교해
예약가능한 프로그램만 출력해 줍니다.

상세정보 페이지 입니다.
해당 프로그램의 상세정보가 출력됩니다.

해당 프로그램의 이미지가 DB와 연동되어 출력됩니다.

DB에 있는 지역의 좌표값을 받아와 구글지도에 
해당 사찰의 위치를 출력해줍니다.

------- 리스트로가서 끊음 --------------

만약 예약하기를 누를 때 로그인이 되어 있지 않다면,
경고창을 출력하고 로그인 페이지로 이동시킵니다.

로그인이 성공할 경우
유저의 편의성을 위해
기존에 예약을 진행하면 프로그램 예약페이지로 이동시켜줍니다.

예약페이지 입니다.
만약, 인원을 선택하지 않고 결제를 진행할 경우
경고창을 출력해줍니다.

----- 예약일 없앰 ------
만약, 예약일이 선택되어 있지 않다면
경고창을 출력해줍니다.

예약 번호를 확인할 수 있습니다.

예약확인 페이지에서 예약된 예약목록을 확인 할 수 있으며,
예약을 클릭하면 해당 예약 상세페이지로 이동합니다.

예약 삭제를 누를 경우 해당 예약번호가 삭제됨을 
확인 할 수 있습니다.

이상입니다.
감사합니다.
